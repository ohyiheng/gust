# Spotify Autotagger

A python utility to auto tag music files using metadata from Spotify, supports auto retrieval of:

- Track Title
- Artist(s)
- Album Title
- Album Artist
- Date
- Track Number
- Total Tracks
- Disc Number
- Total Discs
